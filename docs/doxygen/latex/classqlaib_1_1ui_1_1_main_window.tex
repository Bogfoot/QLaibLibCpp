\doxysubsection{Detailed Description}
Main GUI window\+: drives backends and renders live metrics/plots. 

Responsibilities\+:
\begin{DoxyItemize}
\item select and start the desired backend (live qu\+TAG, replay, mock)
\item pull batches on a timer and update charts/tables
\item manage coincidence pairs and calibration shortcuts 
\end{DoxyItemize}
